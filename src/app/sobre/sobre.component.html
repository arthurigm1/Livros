Sobre o Projeto

O Pagiinova é um e-commerce inovador projetado para revolucionar a compra e venda de livros novos e usados. Nosso objetivo é conectar leitores, editoras e autores em uma plataforma única, oferecendo uma experiência de compra e venda personalizada, segura e eficiente.

Além de facilitar transações, o Pagiinova promove a descoberta de novos títulos, autores e editoras, com recomendações inteligentes e um sistema de avaliações integrado.

Funcionalidades Principais

Compra e Venda

Plataforma intuitiva para comprar e vender livros novos e usados, com filtros avançados e busca por categoria.

Pagamento via PIX

Integração com a API da EFI Pagamentos para geração de QR Code PIX, garantindo pagamentos rápidos e seguros.

Validação de CEP

Utilização da API ViaCEP para validação e busca de CEPs válidos, melhorando a experiência de entrega.

Recomendações Inteligentes

Sistema de recomendações baseado em preferências de leitura e histórico de compras.

Tecnologias Utilizadas

Para garantir um projeto robusto e escalável, utilizamos uma stack moderna e eficiente:

Back-End

Desenvolvido em Java com Spring Boot, garantindo alta performance e segurança.

Front-End

Interface moderna com Angular e estilização com Tailwind CSS.

Banco de Dados

Armazenamento seguro e eficiente com PostgreSQL.

Pagamentos

Integração com a API da EFI Pagamentos para geração de QR Code PIX.

Validação de CEP

Utilização da API ViaCEP para validação e busca de CEPs.

Hospedagem

Back-End hospedado no Railway e Front-End no Netlify.

Desafios e Aprendizados

O desenvolvimento do Pagiinova trouxe diversos desafios técnicos e oportunidades de aprendizado:

Integração com EFI Pagamentos

Implementação da API da EFI Pagamentos para gerar QR Code PIX, garantindo transações seguras e instantâneas.

Validação de CEP com ViaCEP

Integração da API ViaCEP para validação e autocompletar endereços, melhorando a experiência do usuário.

Gerenciamento de Estado

Implementação de um sistema eficiente de gerenciamento de estado global no Angular com RxJS.

Autenticação Segura

Desenvolvimento de um sistema de autenticação e autorização seguro com JWT.

Cada desafio superado contribuiu para o crescimento técnico e a entrega de um projeto robusto e funcional.
